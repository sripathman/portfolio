My Journey in Technology

I began my technical journey in 1996 with a foundation in Windows systems, basic programming, and Microsoft Office. My interest in programming grew, leading me to explore Visual Basic, hardware, HTML, and C++ through a volunteer- run university course in Kaiserslautern, Germany. In 2010, I deepened my programming knowledge with intensive six-week courses in Java and PHP. Building on this foundation, I furthered my web development skills by mastering HTML, CSS, Bootstrap, and the Yii PHP framework.
